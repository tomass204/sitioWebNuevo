Introducción
•	Propósito del documento:
Este manual de usuario está diseñado para guiar a los usuarios de GamingHub, una plataforma comunitaria para gamers, en el uso efectivo de todas las funciones disponibles. Está destinado a usuarios nuevos y existentes que desean maximizar su experiencia en la plataforma, incluyendo navegación, compra de productos, participación en debates y gestión de perfil.

•	Descripción del sistema: 
GamingHub es una aplicación web completa para la comunidad de gamers que ofrece: tienda de productos gaming (accesorios, componentes), noticias actualizadas del mundo gaming, foros de debate, sistema de moderación, y funcionalidades específicas según el rol del usuario (Usuario Básico, Influencer, Moderador, Propietario).

•	Cómo usar este manual: 
Navega por las secciones en orden para usuarios nuevos, o ve directamente a la sección relevante para consultas específicas. Cada sección incluye capturas de pantalla con anotaciones para facilitar la comprensión visual.
 
Requisitos del Sistema
•	Hardware necesario: 
- Computadora o dispositivo móvil con conexión a internet
- Mínimo: Procesador de 1 GHz, 2 GB RAM, 500 MB espacio en disco
- Recomendado: Procesador de 2 GHz o superior, 4 GB RAM o más

•	Software necesario: 
- Navegador web moderno (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Sistema operativo: Windows 10+, macOS 10.15+, Linux Ubuntu 18.04+, Android 8+, iOS 12+
- No se requiere instalación de software adicional

•	Navegación por Pestañas:

- Favoritos: Lista de elementos marcados como favoritos

Solución de Problemas
•	Problemas Comunes:
- Error de carga de imágenes: Verificar conexión a internet y recargar la página
- Error de autenticación: Verificar credenciales y estado del servidor backend
- Problemas con compras: Asegurarse de tener sesión activa y productos en stock
